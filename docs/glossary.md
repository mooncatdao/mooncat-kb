# Glossary

Short definitions for terms used in this repository.

## Canonical Data

Exact source-of-truth data used by tools or downstream references, such as IDs, numeric values, indexes, and mappings.

Status: incomplete. No canonical MoonCat ID or trait datasets have been added yet.

## Curated Data

Data selected, normalized, or grouped by maintainers from one or more sources.

Status: incomplete. Curated categories require source notes before use.

## Source

A reference used to support a fact, category, mapping, or explanation.

Source entries are tracked in `data/sources.json`.

## MoonCat ID

An exact identifier for a MoonCat.

Status: incomplete. This scaffold does not define any MoonCat IDs.

## Trait Mapping

A mapping from MoonCats to visual, metadata, color, category, or other trait labels.

Status: incomplete. This scaffold does not define exact trait mappings.
